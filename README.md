# 图书管理系统 python69

## 项目概述

图书管理系统是一款基于Python开发的Web应用，采用Flask框架和SQLite数据库，用于实现图书信息的增删改查等管理功能。

## 技术栈

- **后端语言**: Python
- **Web框架**: Flask
- **数据库**: SQLite
- **依赖管理**: requirements.txt

## 功能模块

- 用户认证
- 图书信息管理
- 借阅管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- **Python版本**: 3.7
- **依赖库**: sqlalchemy=1.4

## 部署步骤

1. 确保Python 3.7环境已安装。
2. 安装依赖库：`pip install -r requirements.txt`。
3. 根据项目文档进行部署。
